# agent-wiki 知識庫總覽

## 定位

`agent-wiki` 是部門層級的 LLM 驅動知識庫，透過 AgentHQ 入口提供查詢服務。

員工可以用自然語言透過 agent 詢問部門規格、SOP、系統架構等問題，
agent 從這個 Wiki 查詢後整合回答，不依賴個人記憶或散落的文件。

## 知識來源類型

- 系統規格書與 API 文件
- 標準作業程序（SOP）
- 系統架構說明
- 技術知識點與踩坑紀錄
- 對外研究素材（文章、技術報告）

## 使用方式

**人工查詢**：透過 AgentHQ 聊天介面，以自然語言提問
**文件匯入**：將新文件放入 `raw/`，執行 ingest
**定期維護**：執行 lint 確保 wiki 品質
